# minima-dex-widget
A Minima DEX widget website powered by MiniMask for non custodial token swaps payments orderbook and live trading interaction
---

## 🚀 Description

This project demonstrates how to build a decentralized exchange (DEX) style widget using Minima and MiniMask.

Users can:
- Connect MiniMask wallet
- View address and public key
- Check balances and coins
- Create buy/sell orders (UI demo)
- Use a MINIMA / USDT swap interface
- Send payments
- Access live Minima network tools

---

## ⚙️ Features

- MiniMask extension detection
- Wallet connection system
- Address + public key display
- Balance and coins viewer
- DEX-style swap widget
- Buy and sell order UI
- Orderbook (frontend demo)
- Trade history (frontend demo)
- Send payment via MiniMask
- Latest block viewer
- Random generator
- TxPoW lookup
- TxPoW verification
- Install guide
- Security notice
