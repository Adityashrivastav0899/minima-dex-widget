MiniMask Install Guide

  1. Download MiniMask from https://minimask.org/minimask-2.5.zip
  2. Unzip it on desktop
  3. Open Chrome extensions page
  4. Enable Developer Mode
  5. Click Load Unpacked and select the MiniMask folder
  6. Open this website and connect MiniMask

Security Notice

This site never stores private keys and never asks for seed phrases. All signing happens through MiniMask.
